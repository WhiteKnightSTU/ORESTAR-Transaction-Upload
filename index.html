ORESTAR Transaction Export

Pulls Receipts and Payments from Manager that haven't been exported yet, and builds an ORESTAR-compliant XML file.
This runs as a Manager Custom Button — embedded inside Manager, talking to it via its own internal API proxy. No token, URL, or business name to configure; Manager handles all of that automatically. It generates a file for manual upload via ORESTAR's "Upload File" feature — it does not file anything automatically, the candidate or treasurer must still log into ORESTAR to review and file.
